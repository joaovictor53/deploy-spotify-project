# 🎧 Clone Spotify Project

Este é um clone simplificado do Spotify feito com **React**, **Node.js** e **MongoDB**.




## Tecnologias

- React + Vite
- Node.js + Express
- MongoDB
- React Router DOM
- FontAwesome

## ▶ Como rodar localmente

### Backend

cd back-end
npm install
node api/insertMany.js   # Popula o banco
node api/server.js       # Inicia o servidor

#Frontend
cd front-end
npm install
npm run dev
